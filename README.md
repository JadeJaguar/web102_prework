# WEB102 Prework - *Sea Monster Crowdfunding*

Submitted by: **Iman Zahid**

**Sea Monster Crowdfunding** is a website for the company Sea Monster Crowdfunding that displays information about the games they have funded.

Time spent: **4** hours spent in total

## Required Features

The following **required** functionality is completed:

* [x] The introduction section explains the background of the company and how many games remain unfunded.
* [x] The Stats section includes information about the total contributions and dollars raised as well as the top two most funded games.
* [x] The Our Games section initially displays all games funded by Sea Monster Crowdfunding
* [x] The Our Games section has three buttons that allow the user to display only unfunded games, only funded games, or all games.

The following **optional** features are implemented:

* [x] Added a search bar above the games that filters them by name as you type. It's not case sensitive so it works no matter how you type it.
* [x] If your search doesn't match any game, instead of leaving the section blank, a message shows up saying no games were found and it includes whatever you typed.
* [x] Added a sticky nav bar at the top with links that jump to the Welcome, Stats, and Our Games sections. It also scrolls smoothly instead of just teleporting to the section.
* [x] Changed the whole color scheme to a Deep Ocean palette. The background is dark navy, the accents are teal, and the stats cards are a midnight blue color.
* [x] Made the text bigger and changed up the styles. The headings are bolder with teal underlines, the stat labels are uppercase with letter spacing, and the description is in italics now.
* [x] The important numbers in the description (like the total raised, the total number of games, and the unfunded count) are bolded and colored teal so they stand out from the rest of the text.
